# TalentMatch AI

A modern **React** talent matching platform with AI-powered candidate search for recruiters and profile management for candidates.

Dark-themed UI with emerald accents, glassmorphism effects, responsive design and role-based views (Recruiter â†” Candidate).

## Features

- **Two user roles** with separate dashboards
  - Recruiter: AI-powered talent search, candidate comparison, save/contact
  - Candidate: Profile overview, visibility stats, AI improvement suggestions
- **Authentication simulation** (login/signup with role selection)
- **Responsive navigation** with role switcher
- **Beautiful dark UI** using Tailwind CSS + gradients + backdrop-blur
- **Lucide icons** for clean, consistent visuals
- **React Router** for protected routes and navigation
- **Comparison modal** with AI-generated recommendation & scoring table

## Tech Stack

- React 18 (Vite or CRA compatible)
- React Router v6
- Tailwind CSS
- lucide-react (icons)
- Modern CSS

## Usage
- Download the project
- npm install (inside the project folder)
- npm run dev (inside the project folder)
- make sure node js is installed
